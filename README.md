# Streamlit Training

This project is a Streamlit application for collecting and storing survey data. The data is stored in a PostgreSQL database hosted on Render, and the application is deployed on Streamlit Cloud.

## Features

- Collects user input through a form.
- Stores survey data in a PostgreSQL database using SQLAlchemy.
- Provides a user-friendly interface built with Streamlit.
